# Daily Tarot — Claude Code 自動実行仕様

## このリポジトリの目的

毎日、タロットリーディングのHTMLを自動生成し、GitHub Pages で公開する。

公開URL: **https://shiroinu-net.github.io/dailytarot/tarot/**

---

## 毎回実行するタスク

**今日の日付（YYYY-MM-DD）でタロットリーディングを行い、`tarot/index.html` を更新して main にマージする。**

### 手順

> ⚠️ **必須確認（手順 0）**: HTMLを書き始める前に、システムコンテキストの `currentDate` の値を声に出して確認し、その日付を使う。`tarot/index.html` に書かれている既存の日付・`git log` の日付・`date` コマンドの出力は一切使わない。

1. 今日の日付をシステムコンテキストの `currentDate`（日本時間 JST）から取得する。`date +%Y-%m-%d` は UTC を返すため使わない
2. 大アルカナ22枚からランダムに3枚選ぶ（状況・課題・アドバイス）
3. `tarot/index.html` を以下の仕様でHTMLごと上書き生成する
4. **日付の確認（必須）**: HTML 生成後、以下のコマンドで日付が `currentDate` と一致しているか必ず検証する。3箇所すべてに正しい日付が入っていること。

   ```bash
   grep -n "2026-" tarot/index.html
   ```

   確認箇所：`<title>` タグ内、`<div class="date">` 内（スラッシュ区切り）、`<footer>` 内。一箇所でもズレがあれば修正してからコミットする。

5. コミット → フィーチャーブランチにプッシュ → PR作成 → squash merge で main にマージ
6. `claude/tarot-reading-generator-mT5l5` ブランチにも main をマージして push する（GitHub Pages のデプロイ元がこのブランチのため）

### HTML仕様

- **ファイル**: `tarot/index.html` のみ。日付ファイル（`tarot/YYYY-MM-DD.html`）は作らない
- **カード**: 3枚（状況 / 課題 / アドバイス）をランダムに選択
- **正位置・逆位置**: 各カードにランダムで正位置／逆位置を割り当てる（50%ずつ）。逆位置はSVGラッパーに `class="card-svg-wrap reversed"` を付与してCSSで180°回転させる
- **カード表示**: カード名の下に `<div class="card-position upright">正位置</div>` または `<div class="card-position reversed">逆位置</div>` バッジを表示し、さらに `<div class="card-meaning">` でそのカード・その正逆の意味概要を3行程度で表示する
- **テーマ別リーディング**: 仕事・創作活動、体調・メンタル、人間関係、金運・生活 の4テーマ、各テーマで3ポジション分のテキストを書く。カード名の後に `（正位置）` または `（逆位置）` を付ける
- **カード描画**: SVGで描画（各カードを160×260pxのSVGで）
- **デザイン**: ライト背景（`#f9f7f4`）、Noto Sans JP、14px、余白多め
- **強調**: リーディングテキスト内の重要なフレーズは `<strong>` タグで太字にする
- **既存ファイルを参考に**: `tarot/index.html` の現在のスタイル・構造を踏襲する

### Git フロー

```
# フィーチャーブランチで作業
git add tarot/index.html
git commit -m "Daily tarot reading YYYY-MM-DD (カード名1・カード名2・カード名3)"
git push -u origin <branch>

# PR作成 → squash merge → main に反映
# その後、Pages デプロイ用ブランチも更新
git fetch origin claude/tarot-reading-generator-mT5l5
git checkout claude/tarot-reading-generator-mT5l5
git merge main --no-edit
git push origin claude/tarot-reading-generator-mT5l5
git checkout main
```

### 完了条件

- `tarot/index.html` が今日の日付・カードの内容になっている
- main ブランチにマージされている
- `claude/tarot-reading-generator-mT5l5` ブランチにも反映されている
- https://shiroinu-net.github.io/dailytarot/tarot/ に反映される状態

## ユーザープロフィール（リーディングのパーソナライズに使用）

リーディングのテキストを生成する際は、以下のプロフィールを踏まえて内容をパーソナライズすること。

### 基本情報
- **年齢・性別**: 1992年生まれ・女性（30代前半）
- **職業**: 会社員 ＋ 音楽活動（二足のわらじ）
- **居住**: 地方在住、一人暮らし（両親宅が隣接）、犬と同居

### 関心・目標
- **最大のテーマ**: クリエイティブ活動（音楽）を**健康的に、かつ活発に**続けていきたい
- 会社員と音楽活動の両立が日常のコアにある
- 体調・メンタルを整えながら創作エネルギーを維持することが大切

### タロットの使い方
- **読む時間帯**: 朝（一日のはじまりに読む）
- **目的**: 気分のリセット・新鮮な一日を送るための後押し
- ネガティブに引き込むより、前向きに背中を押すトーンを意識する

### テキストを書くときの指針

**仕事・創作活動**
- 「仕事」は会社員としての業務と、音楽活動の両方を指す
- 音楽制作・演奏・表現活動に直接つながるメッセージを含める
- 二足のわらじの疲弊よりも、両者が相乗効果を生む視点を大切に

**体調・メンタル**
- 犬との朝の時間、自然の多い地方生活など、日常の具体的な場面をたまに織り込む
- 「創作を続けるための体づくり・心づくり」という文脈で語る
- 疲れを責めるより、回復と自己ケアを肯定する

**人間関係**
- 家族（両親が隣）との距離感、地方コミュニティ、音楽仲間など多層的な関係がある
- 孤独ではないが、自分の時間・空間を大切にしているタイプ

**金運・生活**
- 会社員の安定収入がベースにある
- 音楽活動への投資（機材・レッスン・ライブ費用など）と生活費のバランス感覚

### 文体・フォーマット
- 重要なフレーズは `<strong>` で太字にする
- 朝に読むことを意識した清々しいトーンを基調にしつつ、**占いとしての深みと緊張感を持たせる**

### 占いとして機能させるための指針（重要）
一般的な前向きコメントは「占いである意味がない」。以下を必ず意識すること：

- **カードの影の側面を使う**: 各カードには光と影がある。例えば女教皇は「内なる知恵」だが同時に「行動しないことへの言い訳」にもなる。戦車は「意志力」だが「暴走・方向違い」も示す。太陽は「喜び」だが「今すぐ出せ」という挑発でもある。
- **問いかける**: 「〜しましょう」より「〜していませんか？」「〜はどうでしょう」のように、読んだ人が自分を見つめ直す問いを投げかける
- **具体的な不快感を含める**: 「耳が痛いけど、確かにそうかも」と感じさせるひと言を各テーマに一つは入れる
- **カード同士の緊張関係を読む**: 女教皇（静・知・待つ）×戦車（動・意志・進む）×太陽（表現・開く・今）のように、3枚の組み合わせが生む物語・矛盾・展開を読み解く
- **励ますのは最後の一文だけでいい**: 前半で本質を突き、最後だけ着地させる構造が読み応えを生む
